# ClassPulse

ClassPulse is a modern Attendance Management System designed to simplify academic administration by providing a centralized platform for attendance tracking, resource sharing, announcements, and marks management.

Built for educational institutions, ClassPulse offers dedicated dashboards for Students, Teachers, and Administrators, ensuring a seamless and organized academic experience.

---

## Features

### Student Portal

- View attendance records and percentages
- Access study materials and resources
- Read announcements from faculty
- View examination marks and academic performance
- Personalized dashboard with academic insights

### Teacher Portal

- Mark and manage attendance
- Upload notes, PDFs, assignments, and study materials
- Create and publish announcements
- Upload and release examination marks
- Manage students within assigned classes

### Admin Portal

- Manage students and teachers
- Create and assign classes
- Import student data using CSV files
- Monitor institution-wide analytics
- Generate attendance and performance reports
